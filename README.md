# Эмулятор ассемблера

## Двухадресная Гарвардская архитектура

### Тест

Сложение:

1) Загрузить массив чисел `1, 2, 3, 4, 5, 0, 0, 0, 0, 0` в поле данных
2) Загрузить код ассемблера
```asm
RESET ACC
LOAD R1 0
ADD ACC R1
LOAD R1 1
ADD ACC R1
LOAD R1 2
ADD ACC R1
LOAD R1 3
ADD ACC R1
LOAD R1 4
ADD ACC R1
STORE ACC 5
```
3) В шестой ячейке памяти должен появится результат суммирования 5 элементов (15)
